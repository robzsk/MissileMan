Missileman
==========
A prototype game written in JavaScript and three.js

[Preview](https://robzsk.github.io/Missileman)

Install
=======
`npm i`

Run
=======
`npm start`


Player Controls
===============

Arrow keys to move left and right

`c` button to morph between man and missile
